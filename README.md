# Nanocat

![Nanocat displaying a MOTD and three messages](screenshot.png)

It's a [Nanochat](https://git.phial.org/d6/nanochat) client!

## Running

1. [Install uv](https://docs.astral.sh/uv/getting-started/installation/)
2. `git clone https://codeberg.org/soxfox42/nanocat.git`
3. `cd nanocat`
4. `uv run main.py SERVER` (replace `SERVER` with your Nanochat server address.)
